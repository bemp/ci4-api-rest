## About CI4 API REST 
Simple project on codeigniter 4

## Requirments

1. PHP >= 8.1
2. MySQL >= 10.3
3. Composer >= 2.2

## Deploy project

1. git clone https://github.com/bemp/ci4-api-rest.git
2. cd ci4-api-rest
3. composer install
4. php spark migrate
5. php spark serve 
8. Open project on http://localhost:8080
